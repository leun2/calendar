## 🛠️ Tech Stack

- **Frontend**: React, TypeScript, CSS
- **Backend**: Spring Boot, Java
- **Authentication**: JWT
- **Database**: JPA

## 📁 Project Structure

calendar/ 
<br>├── client/ # Frontend application
<br>└── server/ # Backend application

## 🚀 Getting Started

### Prerequisites

- Node.js
- Java 17+
- Gradle

### Installation

#### Backend

```
cd server
./gradlew bootRun
```

#### Frontend

```
cd client
npm install
npm start
```
